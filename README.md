# 대명DnP 랜딩페이지

인천 남동구 기반 사인물 제작 전문업체 **대명DnP** 의 소개 랜딩페이지입니다.
간판 · UV인쇄 · 나무현판 · 아크릴 안내판 · LED 간판 · 실사출력 · 장애인 촉지도 안내판 제작.

## 폴더 구성

```
index.html    페이지 본문 (HTML)
style.css     전역 스타일 (리셋 · 반응형)
script.js     동작 스크립트 (FAQ 아코디언)
README.md     설명 파일
images/       페이지에 들어가는 사진 폴더  ← 사진만 바꾸면 됩니다
```

## 열어보기

`index.html` 을 더블클릭하면 브라우저에서 바로 열립니다. (인터넷 연결 시 폰트가 정상 표시됩니다)

## 사진 교체

`images/` 폴더의 파일을 **같은 이름** 으로 실제 사진(png/jpg)으로 덮어쓰면 그대로 반영됩니다.

| 파일명 | 위치 |
|---|---|
| hero.png | 상단 메인 — UV 프린터·작업 현장 |
| product-wood.png | 제작 품목 — 나무현판 |
| product-acrylic.png | 제작 품목 — 아크릴 안내판 |
| product-led.png | 제작 품목 — LED 간판 |
| product-print.png | 제작 품목 — 실사출력·포맥스 |
| product-tactile.png | 제작 품목 — 장애인 촉지도 안내판 |
| product-smartstore.png | 제작 품목 — 스마트스토어 주문 제작 |
| wood-feature.png | 대표 상품 — 나무현판 |

확장자를 .jpg 등으로 바꾸려면 `index.html` 안의 `images/...png` 경로도 함께 수정하세요.

## GitHub Pages 배포

1. 모든 파일을 저장소에 업로드 (`index.html` 은 최상단)
2. **Settings → Pages** 이동
3. Source: `Deploy from a branch`, 브랜치 `main`, 폴더 `/ (root)` → 저장
4. `https://<사용자명>.github.io/<저장소명>/` 으로 접속
